About

IPFS Codex is a web repository for IPFS CIDs and their metadata. This website's purpose is to make the IPFS network more accessible and provide a centralized database to send and receive IPFS CIDs. Users can search for files and contribute to the site's development by submitting new CIDs and metadata.
